# Vision

## Призначення
Micro News Aggregator — мікросервісна платформа для збору, обробки, AI-сумаризації та персоналізованого показу новин з відкритих RSS-джерел.

## Ключові можливості
- Реєстрація / автентифікація користувачів (JWT)
- Автоматичний збір статей з RSS-джерел
- Дедуплікація, тегування і збагачення статей
- AI-сумаризація через LLM
- Персоналізована стрічка та повнотекстовий пошук

## Користувачі
- **Читач** — отримує персоналізовану стрічку, читає сумаризовані статті.
- **Адміністратор** — керує списком RSS-джерел.

## Нефункціональні вимоги
- Відмовостійкість auth-сервісу (дублювання інстансів + Redis для сесій)
- Реплікація MongoDB (replica set)
- Асинхронна обробка через Kafka
